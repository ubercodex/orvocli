# UBER CLI

A minimal, extensible command-line interface built with [Ink](https://github.com/vadimdemedes/ink) and TypeScript — runs natively on **Windows**, **macOS**, and **Linux**.

## Requirements

- Node.js >= 18

## Installation

```bash
git clone https://github.com/ubercodex/uber-cli.git
cd uber-cli
npm install
npm link
```

Then run from anywhere:

```bash
uber
```

## Development

```bash
npm run dev   # run with file watching
npm start     # run once
```
